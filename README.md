# minecraft-player-info-api
Provides python functions that query mojang player API

**Install guide**
##Step 1:
install git (if not already installed) [git install](https://git-scm.com/install/)
install python (if not already installed) [python install](https://www.python.org/downloads/)
##Step 2:
Type the following into your terminal and hit enter
```git clone https://github.com/bluecactus404/minecraft-player-info-api.git```
then type
```cd minecraft-player-info-api```
then
edit the .env file in any text editor and replace YOUR_APP_NAME_HERE text with your app's name as a courtasy to mojang while keeping the quotation marks. Save and exit the editor.
##Step 3:
To run the file, type the following into your terminal and hit enter
```python3 main.py```
To quit: hit ```ctrl + c```

##To test:
open a text editor and delete the """ marks at the bottom of the file. Replace your_account_name_here with your minecraft account name save the changes and run the file.
